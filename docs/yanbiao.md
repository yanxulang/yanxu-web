# 言标模板语言

言标是随言枢发布的中文 HTML 超集。普通 HTML 保持原样；动态插值和控制命令使用独立定界符。

## 插值与路径

```html
<h1>{{ 页面.标题 }}</h1>
```

路径用`.`读取嵌套典。不存在的路径得到空值并输出空串。普通插值始终 HTML 转义，适用于元素文字和带引号的属性值。

支持的过滤器：

```html
{{ 名称|默认:匿名 }}
{{ 名称|大写 }}
{{ 名称|小写 }}
{{ 文章列|长度 }}
{{ 数据|JSON }}
```

过滤器从左到右应用。`JSON`结果仍经过 HTML 转义；它适合展示，不应用来生成可执行脚本。

## 条件

```html
{% 若 已登录 %}
  <p>欢迎，{{ 用户.姓名 }}</p>
{% 否则 %}
  <p>请登录</p>
{% 终若 %}

{% 若 非 已归档 %}...{% 终若 %}
```

空、假、数值零、空文字、空列和空典是假值，其余值为真。

## 循环

```html
{% 逐 文章 于 文章列 %}
  <article data-index="{{ 循环.索引 }}">
    <h2>{{ 文章.title }}</h2>
  </article>
{% 终逐 %}
```

`循环`提供`序号`（从 1 开始）、`索引`（从 0 开始）、`首项`和`末项`。循环可以嵌套；子作用域查不到名称时会读取父作用域。

## 包含

```html
{% 含 partials/header.yb.html %}
```

被包含模板继承当前作用域。模板名不加引号，必须是模板根内以`.yb.html`结尾的安全相对路径。

## 继承与块

基础模板：

```html
<!doctype html>
<main>{% 块 主体 %}<p>默认内容</p>{% 终块 %}</main>
```

子模板：

```html
{% 承 base.yb.html %}
{% 块 主体 %}<h1>{{ 标题 }}</h1>{% 终块 %}
```

一个模板只能声明一个`承`。子模板用同名块替换基础模板默认块，多层继承保留更下层的同名覆盖。

## 注释

```html
{# 这段内容不会输出 #}
```

## 显式可信 HTML

```yanxu
定 数据 为 {「正文」：言枢.言标.信任HTML（已审计源码）}；
```

只有`言标安全HTML`实例会绕过插值转义。该工厂是信任升级和代码审查标记，不执行清洗。不得传入用户输入或来源不明的数据。

## 资源与路径限制

- 单模板源码最大 256 KiB；
- 包含/继承链最大 16 层；
- 循环包含或继承会报`YANBIAO_CYCLE`；
- 绝对路径、反斜杠、NUL、空段、`.`和`..`会报`YANBIAO_PATH`；
- 未闭合定界符、控制块和未知过滤器使用`YANBIAO_*`稳定错误前缀。

言标不是浏览器 HTML 解析器，也不清洗 CSS、JavaScript 或 SVG。CSP、URL 业务规则和可信内容来源仍由应用负责。
