# yanxu-web API 参考

入口模块是`src/言序Web.yx`，当前公开版本常量为`版本 = "0.1.0"`。

## 路由

### `Web路由`

只读域：`方法`、`模式`、`处理器`。方法必须是合法 HTTP token，模式必须以`/`开头。

### `Web路由器`

`添加（方法，模式，处理器）`注册路由并返回自身；相同方法与模式不能重复。应用通过它区分：

- 方法和路径命中：执行处理器；
- 路径命中但方法不匹配：返回 405 与`Allow`；
- 没有路径命中：返回 404。

模式段支持字面量、`:名称`单段参数和末尾`*名称`剩余路径参数。通配段一旦出现就消费余下路径。

## `Web上下文`

| 域/成员 | 说明 |
| --- | --- |
| `请求` | 底层`HTTP请求`。 |
| `路径参数` | 当前路由匹配得到的参数典。 |
| `状态` | 应用共享状态典。 |
| `允许` | 路径命中时允许的 HTTP 方法列。 |
| `参数值（名称）` | 读取单个路径参数。 |
| `查询值（名称）` | 读取第一个查询值。 |
| `首部值（名称）` | 不区分大小写读取请求首部。 |
| `Cookie值（名称）` | 读取请求 Cookie。 |
| `状态值（名称）` | 读取应用共享状态。 |

## `Web应用`

只读公开域为`路由器`和`状态`。

| 成员 | 返回 | 说明 |
| --- | --- | --- |
| `路由（方法，模式，处理器）` | `Web应用` | 注册任意方法。 |
| `取（模式，处理器）` | `Web应用` | 注册 GET。 |
| `发（模式，处理器）` | `Web应用` | 注册 POST。 |
| `使用（中间件）` | `Web应用` | 追加`法（上下文，下一步）`。 |
| `设错误处理器（处理器）` | `Web应用` | 设置`法（上下文，所误）`。 |
| `挂静态（网址前缀，文件根）` | `Web应用` | 注册末尾通配 GET 路由。 |
| `处理（请求）` | `HTTP响应` | 路由、执行中间件链并验证返回类型。 |

`应用（）`是`Web应用`工厂。

## 响应

| 工厂 | 说明 |
| --- | --- |
| `HTML响应（节点）` | 安全渲染 HTML 节点并返回 200。 |
| `HTML状态响应（状态码，节点）` | 安全渲染节点并指定状态码。 |
| `JSON响应（值）` | 交给`yanxu-http` JSON 响应。 |
| `言据响应（正文）` | 返回已序列化的言据文字。 |
| `文字响应（正文）` | 返回 UTF-8 纯文字。 |
| `字节响应（正文）` | 返回通用二进制。 |
| `状态响应（状态码，正文）` | 返回指定状态的纯文字。 |

所有处理器、中间件和错误处理器最终都必须返回`HTTP响应`。

## 静态文件

- `静态文件响应（文件根，相对路径）`：验证相对路径、读取普通文件、推断媒体类型并设置`Cache-Control: no-cache`；
- `静态处理器（文件根）`：返回读取`path`通配参数的路由处理器；路径或文件失败统一为 404。

## `Web开发服务器`

只读域为`应用`和`地址`，可配置`接受超时毫秒`，默认 1000。

| 成员 | 说明 |
| --- | --- |
| `运行若干（请求数）` | 监听并处理指定数量的请求；`0`表示持续运行。 |
| `运行（）` | 持续运行。 |

`开发服务器（应用，地址）`是构造工厂。服务器串行接受连接，每个连接读取一个请求、发送一个响应并关闭。架构细节见[请求生命周期](architecture.md)。
