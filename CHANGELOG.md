# 更新记录

本项目遵循语义化版本。`1.x`保持已记录的公开名称、签名、错误代码和默认安全策略兼容。

## 1.0.0 - 2026-07-18

### 新增

- 增加请求与响应预算、规范言据正文、JSON/言据内容协商和稳定错误详情。
- 增加可注入会话存储、会话轮换与销毁、同源双提交 CSRF 防护。
- 增加静态缓存验证器、条件请求、单范围响应和隐藏文件策略。
- 增加显式 OpenAPI 元数据、隔离快照、提供器协议和 JSON 端点。
- 扩展无端口测试客户端，并增加言访自定义测试传输。
- 增加无端口示例、路由/言访基准和完整 1.0 文档门禁。

### 兼容性

- 默认模块和公开言标子模块进入 1.0 稳定兼容面。
- 最低言序版本为 1.1.12，依赖稳定版 yanxu-http、yanxu-html、yanju 和 yanxu-request。
- 保留`Web应用`、`Web上下文`、`Web开发服务器`、`应用`和`开发服务器`兼容入口。

### 工程

- 固定完整传递依赖锁图，并验证离线重建不漂移。
- 示例和基准作为独立消费者包，通过公开导出与各自锁图验证言枢 1.0。
- CI 覆盖格式、静态检查、机器 API、14 项规格、双执行器对照、示例、基准、跨平台构建和新增元数据。

## 0.2.1 - 2026-07-15

- 升级到言序清单/锁文件格式 2 与最低言序 1.1.5。
- 显式导出默认言枢入口与`包:yanxu-web/言标`子模块，构建前固定完整传递依赖图。
- 迁移到纯言序言包 0.2.0 工作流和言序 1.1.5 CI 基线。

## 0.2.0 - 2026-07-15

- 产品名更新为“言枢”，保留`yanxu-web`技术包标识和 0.1 兼容 API。
- 加入集中配置、命名路由、反向 URL、命名空间路由组和常用 HTTP 方法。
- 加入可替换 404/405、模板响应、重定向、Cookie、空响应和 JSON 请求正文。
- 加入无需打开端口的`言枢测试客户端`和框架默认安全首部中间件。
- 内建“言标”HTML 超集：自动转义、条件、循环、过滤器、包含、模板继承与显式可信 HTML。
- 将模板路径、源码大小、包含深度、循环包含和危险路径纳入稳定错误边界。

## 0.1.0 - 2026-07-14

- 建立应用、路由器、路径参数和请求上下文。
- 加入洋葱式中间件链和统一错误处理器。
- 加入 HTML、JSON、言据、文字与字节响应门面。
- 加入防目录穿越的静态文件服务和串行本地开发服务器。
